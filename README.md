# СТВ Life — прототип платформы вовлечения

Кликабельный прототип внутренней платформы вовлечения сотрудников **СТВ Life** в фирменном стиле СТВ. Демо для защиты концепции.

**Живая версия:** https://khinkis13-rgb.github.io/stv-life-prototype/

## Что внутри
- **Главная** (`index.html`) — баланс СТВ Coins, активности, каталог, рейтинг
- **Личный кабинет** (`cabinet.html`) — баланс, история, бейджи, заказы
- **СТВ Store** (`store.html`) — магазин мерча за Coins
- **СТВ Lab** (`lab.html`) — идеи и инициативы сотрудников
- **СТВ Vibe** (`vibe.html`) — мероприятия и фотоотчёты

Интерактив на чистом JS: баланс Coins переносится между экранами, форма подачи идеи, обмен Coins, запись на активности. Внизу страницы — «Сбросить демо».

Статичный сайт без сборки. Данные — демонстрационные (вымышленные).
